# Updash
Updash modular utility

# Summary
Modular utility functions

# Installation
```
npm install updash
```

# Usage
```
var up = require('updash');

var primary = [1,2,3,4];
var secondary = [2,3];

var result = up.isSubArray(primary, secondary);

console.log(result);
```
Output: true

# API
- isSubArray  (determines if an array is a sub array of another array)
